1. Примечания к документу "Пример ПМИ"

   1.1. Поскольку данный документ предназначен для ознакомительного просмотра, список подсистем и, следовательно, список проверяемых функций значительно сокращены. В приложении А приведены проверки только для подсистемы «Реестр обращений», хотя в таблице 3 представлены также подсистемы «Создание и редактирование карточки обращения» и «Подготовка ответа на обращение». Это сделано для сохранения наглядности документа: если в Системе три подсистемы, то логично привести пункты программы, касающиеся всех трех подсистем. Однако, как уже указано ранее, методика проверки выполнения требований к функциям (задачам), выполняемым подсистемами Системы, приведена только для подсистемы «Реестр обращений» в целях сокращения объема документа.

   1.2. В документе использованы стили из стайлгайда заказчика. Стили, которые использовались в компании, можно посмотреть в документе "Инструкция по подготовке технической документации".
